# Source footage note

`video-source/blackstire-reel.mp4` (not committed — see `.gitignore`) is not real Black's Tire footage. It's a locally-compiled pan/zoom/crossfade reel built with `ffmpeg` from six real stock photographs already used elsewhere on this site (technician, engine bay, tire tread, garage, service bay) — a free, offline alternative to paid AI video generation (Veo/Seedance/etc. via inference.sh's `belt` CLI, which requires a funded account this project doesn't have).

The `animated-website` skill doesn't care how the source video was produced — it just needs real motion to scrub through. This is genuine video, genuinely free, genuinely offline. It is not AI-generated and not real shop footage.
